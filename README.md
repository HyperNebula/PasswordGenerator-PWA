# PasswordGenerator PWA

[![Live Demo](https://img.shields.io/badge/demo-online-green.svg)](https://hypernebula.github.io/PasswordGenerator-PWA/)

A simple Progressive Web App that can be installed on a device and works offline. Genereates a 24 character password that can be copied from the webpage.

**Live Demo:** [hypernebula.github.io/PasswordGenerator-PWA/](https://hypernebula.github.io/PasswordGenerator-PWA/)
